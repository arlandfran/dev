# arland.dev

My personal website

## Tech stack

- [Astro](https://astro.build) - Web Framework
- [TailwindCSS](https://tailwindcss.com) - Utility CSS Framework

## Running locally

Clone repository

```sh
git clone https://github.com/arlandfran/dev
```

Install dependencies

```sh
pnpm install
```

Run development server

```sh
pnpm dev
```
